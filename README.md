# SNUDM Technical Reports

### Usage
1. `korean.tex`를 연다.
1. `korean.tex` 내에서 "TODO"라고 표시된 줄의 내용을 변경한다.
    - 일련번호: SNUDM-TR-[year]-[two digit number] (ex: SNUDM-TR-2015-01)
